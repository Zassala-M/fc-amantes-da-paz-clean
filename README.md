## Sobre o projeto

Este projeto é uma aplicação web construída com React, TypeScript, Vite e Tailwind CSS.  
Ele serve como base para desenvolvimento de interfaces modernas e responsivas utilizando boas práticas de programação.

## Tecnologias usadas

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Como editar o projeto

Você pode trabalhar no projeto usando seu editor de código preferido:

```sh
# 1. Clone o repositório
git clone <SEU_GIT_URL>

# 2. Navegue até o diretório do projeto
cd <NOME_DO_PROJETO>

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
